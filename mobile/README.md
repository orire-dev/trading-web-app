# Trading App - React Native Mobile

A simple, beginner-friendly React Native mobile app for eToro trading.

## Design Principles for Novice Users

Based on best practices for trading apps:

1. **Visual Clarity**: Large, clear numbers and simple charts
2. **Color Coding**: Green = profit, Red = loss, Blue = neutral
3. **Progressive Disclosure**: Show basics first, details on demand
4. **Guided Actions**: Step-by-step workflows for trading
5. **Educational Tooltips**: "What does this mean?" buttons everywhere
6. **One-Tap Actions**: Minimize clicks for common tasks
7. **Visual Feedback**: Immediate confirmation of actions

## Key Features

- 📊 Simple dashboard with big, readable numbers
- 🎯 Goal setting with visual progress bars
- 📈 Basic charts (no overwhelming technical analysis)
- ✅ One-tap signal approval
- 📚 In-app learning with simple explanations
- 🎨 Calm, uncluttered design

## Tech Stack

- React Native (Expo)
- React Navigation
- Reanimated for smooth animations
- React Native Paper (Material Design) for simple components
- Socket.IO client for real-time updates
